Privacy Policy for Vocabulary Recall

Vocabulary Recall is a Chrome browser extension that helps users remember vocabulary through gentle recall prompts while browsing.

Data Collection and Use

Vocabulary Recall does not collect, store, or transmit any personal data.

All vocabulary words, learning progress, settings, and preferences are stored locally on the user's device using Chrome’s local storage. This data is used solely to provide recall reminders, daily progress summaries, and user-configurable settings.

Third-Party Services and AI Generation

If users explicitly request AI-generated content (such as definitions or example sentences), only the selected vocabulary word is sent to an external AI service. No browsing activity, personal data, or user content is transmitted or stored by the AI service.

No Tracking or Analytics

Vocabulary Recall does not use analytics, tracking, advertising services, or cookies.

Changes to This Privacy Policy

This privacy policy may be updated if new features are introduced that affect how data is handled. Any updates will be reflected on this page.

Contact

If you have any questions about this privacy policy, please contact the developer via the Chrome Web Store listing.
